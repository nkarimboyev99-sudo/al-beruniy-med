# Lab Registr - Tibbiyot Ro'yxatga Olish Tizimi

Tibbiy laboratoriya va klinikalar uchun bemorlarni ro'yxatga olish, diagnoz qo'yish va hisobot tizimi.

## Texnologiyalar

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas

## Loyiha Strukturasi

```
Registr/
├── backend/          # Node.js API server
│   ├── models/       # MongoDB modellar
│   ├── routes/       # API routes
│   ├── middleware/   # Auth middleware
│   └── server.js     # Entry point
│
└── frontend/         # React SPA
    └── src/
        ├── pages/    # Sahifalar
        └── App.jsx   # Main component
```

## Deploy

- **Frontend**: Netlify
- **Backend**: Render.com
- **Database**: MongoDB Atlas
